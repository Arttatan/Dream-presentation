КАК ПОЛОЧИТЬ ЧЁТКИЙ ФОН (экран 01)
==================================

Сейчас размытие и «двойной текст» — потому что в public/scenes/scene01.png
лежит маленький кусок коллажа (305×304) С УЖЕ НАПЕЧАТАННЫМ текстом.

Нужен другой файл:

1. Экспортируйте из ChatGPT / Figma / Photoshop ТОЛЬКО фон:
   - космос, луна, облака, силуэт
   - БЕЗ вопроса, БЕЗ кнопок, БЕЗ «01»

2. Размер: минимум 1920 × 1080 (лучше 2560 × 1440)

3. Сохраните как:
   public/scenes/scene01.png
   (или scene01.webp)

4. Обновите страницу: Ctrl+F5

Пока файла нет или он меньше 1280px — фон остаётся тёмным,
текст и кнопки только из HTML (без «призрака»).
