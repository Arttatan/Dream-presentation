ЧЁТКИЕ ФОНЫ (обязательно для качества как в дизайне)
====================================================

Коллаж из 12 картинок слишком маленький — растягивать его нельзя.

Что нужно от вас для сцены 01:
1. Экспортируйте ТОЛЬКО фон слайда «Вопрос для зала» из ChatGPT/Figma/Photoshop.
2. Размер: минимум 1920×1080 (лучше 2560×1440).
3. Без текста и кнопок — только картинка (космос, луна, силуэт).
4. Сохраните как PNG или JPG.

Положите файл, например: assets\incoming\01.png

Затем в терминале:
  npm run import-bg -- 01 assets/incoming/01.png

Обновите страницу в браузере (Ctrl+F5).

Файл появится здесь: public\backgrounds\hq\scene-01.webp

Повторите для слайдов 02, 03 … (номер = номер сцены).
