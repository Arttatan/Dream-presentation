# PresentDream — план интерактивной презентации

## 1. Карта сцен (13 экранов)

| № | ID | Навигация | Заголовок сцены | Шагов reveal | Фон / атмосфера |
|---|-----|-----------|-----------------|--------------|-----------------|
| 01 | `quiz` | Вопрос для зала | Вопрос для зала | 5 | Космос, луна, звёзды, туман |
| 02 | `intro` | Вступление | Вступление: почему я выбрал тему «Сны» | 4 | Комната → сон за окном |
| 03 | `ancient` | Древняя загадка | Сны — древняя загадка | 4 | Пустыня, таблички, пирамиды |
| 04 | `theory-1` | Теория 1 | Теория 1: сны исполняют желания | 3 | Профиль Фрейда, небула подсознания |
| 05 | `theory-2` | Теория 2 | Теория 2: сны помогают запоминать | 4 | Светящийся мозг, «полароиды» |
| 06 | `theory-3` | Теория 3 | Теория 3: сны помогают забывать | 4 | Растворяющиеся файлы, пыль |
| 07 | `theory-4` | Теория 4 | Теория 4: сны не дают мозгу «застыть» | 3 | Экраны-воспоминания, силуэт |
| 08 | `theory-5` | Теория 5 | Теория 5: сны — тренировка на опасность | 4 | Тёмный лес, волк, туман |
| 09 | `theory-6` | Теория 6 | Теория 6: сны помогают пережить стресс | 4 | Спокойный океан, закат |
| 10 | `theory-7` | Теория 7 | Теория 7: сны решают задачи | 4 | Бензол, книга, змея |
| 11 | `lucid` | Осознанные сны | А что со сновидениями «на заказ»? | 2 | Портал, рука, символы |
| 12 | `finale` | Итог | Итог + ответ на викторину | 8 | Галактика, 7 иконок теорий |
| 13 | `audience` | Вопрос в зал | Вопрос в зал (в конце) | 5 | Объединение всех тем, мягкий свет |

**Логика reveal на каждой сцене:** заголовок → абзац 1 → абзац 2 → … → (сцена 01: интерактивные варианты) → подсказка «Нажмите Enter…».

**Текст:** дословно из согласованного доклада, без сокращений. Разбит на chunks ≤ ~280 символов / ≤ 5 строк.

## 2. Структура компонентов

```
src/
├── main.tsx
├── App.tsx
├── index.css
├── types/
│   └── presentation.ts
├── data/
│   └── scenes.ts              # весь текст + метаданные сцен
├── engine/
│   ├── PresentationEngine.tsx # корневой оркестратор
│   ├── SceneController.tsx    # рендер одной сцены
│   ├── KeyboardNavigation.ts  # хук: Enter, Space, ↑↓, Esc
│   ├── RevealText.tsx         # пошаговый показ текста
│   ├── BackgroundAnimator.tsx # выбор и parallax фона
│   └── ProgressManager.ts     # localStorage + расчёт %
├── backgrounds/
│   ├── Particles.tsx
│   ├── SceneBackgrounds.tsx   # 13 кинематографичных слоёв
│   └── index.ts
└── components/
    ├── SceneChrome.tsx        # номер, меню, прогресс-точки
    ├── ProgressIndicator.tsx
    ├── ContinueHint.tsx
    ├── SectionMap.tsx         # Esc — карта разделов
    ├── QuizOptions.tsx
    └── RichText.tsx           # **жирный** → highlight
```

### Ответственность модулей

| Модуль | Роль |
|--------|------|
| **PresentationEngine** | `currentScene`, `currentRevealStep`, `advance` / `retreat`, `goToScene`, флаг `isSceneComplete`, quiz state |
| **SceneController** | Layout сцены: chrome + BackgroundAnimator + RevealText + ContinueHint |
| **KeyboardNavigation** | Глобальные слушатели; блокировка при открытой карте |
| **RevealText** | Framer Motion: fade + translateY; только текущий chunk |
| **BackgroundAnimator** | Parallax по `mousemove`, particles, scene-specific gradients |
| **ProgressManager** | `localStorage`: sceneIndex, revealStep, quizAnswer |

## 3. Wireframe (каждый экран)

```
┌─────────────────────────────────────────────────────────────┐
│ 01  Вопрос для зала                              ≡  (Esc)  │
│ ● ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○  (вертикальные точки слева)      │
│                                                             │
│   [Крупный заголовок, ключевое слово — #8879FF]             │
│                                                             │
│   [Текущий блок текста — max 4–5 строк]                     │
│   [fade-in, лёгкий сдвиг вверх]                             │
│                                                             │
│                    [ЖИВОЙ ФОН — 70% кадра]                  │
│                    космос / лес / мозг …                    │
│                                                             │
│              Нажмите Enter или кликните для продолжения     │
└─────────────────────────────────────────────────────────────┘
```

- **Лево:** текст (~40% ширины на desktop).
- **Право/центр:** визуальная метафора сцены (CSS + canvas particles).
- **Мобильный:** текст сверху, фон снизу, те же пропорции воздуха.

## 4. Анимации

| Элемент | Поведение |
|---------|-----------|
| Смена сцены | crossfade 0.8s, фон scale 1.02→1, opacity 0→1 |
| Reveal шаг | text: opacity 0→1, y: 16→0, duration 0.55s, ease [0.22, 1, 0.36, 1] |
| Фон | particles drift; слои parallax ±8px от курсора; «дыхание» glow 4s loop |
| Quiz кнопки | hover glow #5348FF; selected — border + pulse |
| Continue hint | появление после последнего шага; opacity pulse 2s |
| Section map | Esc: backdrop blur, список 13 сцен, stagger 0.04s |
| Переход назад | тот же reveal, шаг уменьшается без анимации «отмены» текста |

## 5. Управление

| Действие | Результат |
|----------|-----------|
| Click / Enter / Space / ↓ | следующий reveal; если сцена завершена → следующая сцена |
| ↑ | предыдущий reveal или предыдущая сцена |
| Esc | карта разделов (jump) |
| Quiz | выбор A–Г сохраняется в state + localStorage |
